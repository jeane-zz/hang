组件注册

注册语法糖: component()函数会自动调用extend(), 所以将extend部分直接写入到component的第二个参数中

	Vue.component = ('component',{
		template:`
			<div> 
				通过语法糖注册的组件 
			</div> 
		`,
	})

使用script标签或template标签，将组件内容分离出来。

    这样做的好处是：
  • 可以避免拼接字符串
  • 降低html和JavaScript的耦合性

使用script标签：注意修改script标签默认的type属性，告诉浏览器这不是js脚本，则浏览器在解析html文档时会忽略该script标签内的内容

	<script type="text/x-template" id="comp_script">
		<div class="test">
			这是通过<b>script标签</b>添加的组件
		</div>

	</script>

	Vue.component('compscript', {
		template: '#comp_script'
	})

使用template标签：不需要知道type，只指定id就可以了

	<template id="comp_template">
		<div class="test">
			这是通过<b>template标签</b>添加的组件
		</div>

	</template>

	Vue.component('comptemplate', {
		template: '#comp_template'
	})
